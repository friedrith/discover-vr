# Oculus quest

## How to upload an app

Download [adb](https://www.xda-developers.com/install-adb-windows-macos-linux/)

```bash
# first installation
$ ./platform-tools/adb install Base/Build.apk

# second installation
$ ./platform-tools/adb uninstall ca.etsmtl.oculus
$ ./platform-tools/adb install Base/Build.apk


```
